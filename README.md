# State Machine App

This app show a simple state machine hook implantation by example of traffic light.

(react + ts + vitest)

[online site](https://yanai101.github.io/trafficLigth.github.io/)

## Also include

- Testing for the app.

- Testing for the machine hook.

### install and run

- Install and run

```
npm i
npm run dev
```

- testing

```
npm test
```
